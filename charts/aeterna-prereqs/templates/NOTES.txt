=== Using with the Aeterna chart ===

Install the main Aeterna chart with connection parameters:

  helm install aeterna charts/aeterna \
    --set postgresql.host={{ include "prereqs.postgresql.host" . }} \
    --set postgresql.existingSecret={{ include "prereqs.postgresql.secretName" . }} \
    --set redis.host={{ include "prereqs.redis.host" . }} \
{{- if .Values.vectorStore.qdrant.enabled }}
    --set vectorStore.host={{ include "prereqs.vectorStore.host" . }} \
    --set vectorStore.port={{ include "prereqs.vectorStore.port" . }}
{{- end }}
